Project Description

We will be implementing a custom BIOS that will complete the necessary basic functions:

  • Checking all device drivers upon start up
  • Power On Self Test (POST)
  • Loading into the bootloader (GRUB, Windows Bootloader, etc.)
  • Inclusion of a basic user interface
